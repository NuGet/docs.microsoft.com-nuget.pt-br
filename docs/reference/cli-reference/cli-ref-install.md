<span data-ttu-id="87a97-113">em que `<packageID>` nomeia o pacote a ser instalado (usando a versão mais recente) ou `<configFilePath>` identifica o arquivo de `packages.config` que lista os pacotes a serem instalados.</span><span class="sxs-lookup"><span data-stu-id="87a97-113">where `<packageID>` names the package to install (using the latest version), or `<configFilePath>` identifies the `packages.config` file that lists the packages to install.</span></span> <span data-ttu-id="87a97-114">Você pode indicar uma versão específica com a opção `-Version`.</span><span class="sxs-lookup"><span data-stu-id="87a97-114">You can indicate a specific version with the `-Version` option.</span></span>
