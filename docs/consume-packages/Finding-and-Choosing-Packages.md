## <a name="search-syntax"></a><span data-ttu-id="314a1-198">Sintaxe de pesquisa</span><span class="sxs-lookup"><span data-stu-id="314a1-198">Search Syntax</span></span>

<span data-ttu-id="314a1-199">A pesquisa de pacote do NuGet funciona da mesma forma no nuget.org, na CLI do NuGet e dentro da extensão do Gerenciador de Pacotes do NuGet no Visual Studio.</span><span class="sxs-lookup"><span data-stu-id="314a1-199">NuGet package search works the same on nuget.org, from the NuGet CLI, and within the NuGet Package Manager extension in Visual Studio.</span></span> <span data-ttu-id="314a1-200">Em geral, a pesquisa é aplicada a palavras-chave, bem como descrições de pacote.</span><span class="sxs-lookup"><span data-stu-id="314a1-200">In general, search is applied to keywords as well as package descriptions.</span></span>
