## <a name="usage"></a><span data-ttu-id="f088b-112">Uso</span><span class="sxs-lookup"><span data-stu-id="f088b-112">Usage</span></span>

<span data-ttu-id="f088b-113">em que `<packageID>` nomeia o pacote de instalação (usando a versão mais recente), ou `<configFilePath>` identifica o `packages.config` arquivo que lista os pacotes sejam instalados.</span><span class="sxs-lookup"><span data-stu-id="f088b-113">where `<packageID>` names the package to install (using the latest version), or `<configFilePath>` identifies the `packages.config` file that lists the packages to install.</span></span> <span data-ttu-id="f088b-114">Você pode indicar uma versão específica com o `-Version` opção.</span><span class="sxs-lookup"><span data-stu-id="f088b-114">You can indicate a specific version with the `-Version` option.</span></span>
